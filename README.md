# DualSubStream (Modified Version)

- **Author:** harutopia0
- **Original Creator:** Deshan Nawanjana

## Description
This is a customized version of SubStream that supports uploading multiple subtitles simultaneously and provides more flexible subtitle management. Special thanks to Deshan Nawanjana for creating the original amazing platform.

## Features
- Upload up to 2 subtitle files simultaneously (suitable for bilingual users).
- Customized subtitle list interface.
- Preserves the original time synchronization and UI customization features.

---

## Installation & Setup

### Step 1: Clone the Repository

Open your terminal or command prompt and run the git clone command with the repository URL:

```bash
git clone https://github.com/harutopia0/dualsubstream
```

*(Note: If you do not have Git installed, you can click the green Code button on the GitHub repository page, select Download ZIP, and then extract the downloaded archive).*

### Step 2: Load into Chrome

1. Open Google Chrome and navigate to: chrome://extensions/
2. Turn on Developer mode using the toggle switch in the top-right corner.
3. Click the Load unpacked button located in the top-left corner.
4. Select the folder containing this source code (the directory where manifest.json is located).

---

## How to Use

1. Click on the Extension icon on the Chrome toolbar to open DualSubStream.
2. Upload your subtitle files (Supports up to 2 files simultaneously for bilingual display).
3. Customize the subtitle list, adjust time synchronization, or change the UI styling as needed.
