# DualSubStream (Modified Version)

- **Author:** harutopia0
- **Original Creator:** Deshan Nawanjana

## Description
This is a customized version of SubStream that supports uploading multiple subtitles simultaneously and provides more flexible subtitle management. Special thanks to Deshan Nawanjana for creating the original amazing platform.

## Features
- Upload up to 2 subtitle files simultaneously (suitable for bilingual users).
- Customized subtitle list interface.
- Preserves the original time synchronization and UI customization features.
